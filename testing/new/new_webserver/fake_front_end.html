Transport API – Fake Front End

Send real HTTP requests to the back end and inspect responses
→ http://localhost:8080
GET

MapService

/map

Returns all stop nodes and transport metadata (weather, messages, warnings) for the given user location.

GET

MapService (cached)

/map?cached=true

Client already has nodes cached locally — returns only the information field (no nodes array). Uses ?cached=true?lat=…?long=… separator syntax as per spec.

GET

TimetableService – TimetableLookup

/timetable

Returns timetable data (with live tracking updates where available) for up to 10 nodes from a given departure time.

GET

TimetableService – ServiceLookup

/services/:nodeId

Returns the services (bus lines / trains) that stop at the given node, including their graphical map paths.

GET

TimetableService – LiveTracking

/tracking/:nodeId

Returns live GPS positions of vehicles currently approaching or having just left the given node.

GET

RouteService

/routes

Computes a selection of routes (fastest / fewest-changes / least-walking) between two nodes at a given departure time.
